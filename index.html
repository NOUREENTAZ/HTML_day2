2. Explain how many different types of lists we have in html

There are 3 types of lists

  • Ordered List - have an order which is either numeric or alphabetical order it will place whithin li tag and start with ol tag..
    Syntax
    <ol>
    <li> Item 1 </li>
    <li> Item 2 </li>
    <li> Item 3 </li>
    </ol>

    numeric means 1,2,3... & alphabetical means A,B,C .... or a,b,c....
    default value is number.

    like
    <ol type ="1" >
    <li> Item 1 </li>
    <li> Item 2 </li>
    <li> Item 3 </li>
    </ol>
    ex-
    1. Item 1
    2. Item 2
    3. Item 3

    various way to use a ordered list in below

  • type = "1" : List will start from 1.
  • type = "A" : Here the list will start from A.
  • type = "a" : Here the list will start from lowercase a.
  • type = "I" : The list will start from Roman numbers.
  • type = "i": TheList will start from lowercase Roman numbers


    • -----------------------------------------------------------------
  • Unorder List - have an unorder which unorder list it will bullet or unordered format it will place whithin li tag and start with ul tag..
    Syntax
    <ul>
    <li> Item 1 </li>
    <li> Item 2 </li>
    <li> Item 3 </li>
    </ul>

    circle means o,o,o ...
    default value is disc.

    like
    <ul type ="circle" >
    <li> Item 1 </li>
    <li> Item 2 </li>
    <li> Item 3 </li>
    </ol>
    ex-
    • Item 1
    • Item 2
    • Item 3

    various way to use a ordered list in below

  • type = "circle" : It gives a circle list item marker.
  • type = "square" : It gives square as list item marker.
  • type = "disc" : This is the default filled circular bullet item marker.
  • type = "none" : This is used to unmark list items.
    • ----------------------------------------------------

  • Discription List

    A description list is a type of list where each item has a description. It is also known as a definition list.
    The <dl> tag is used to create description list,
    the <dt> tag defines the type, and the
    <dd> tag describes each item in data.


    Syntax
    <dl>
    <dt> Item 1 </dt>
    <dd> Item 2 </dd>
    </dl>
    Ex-
    Code
    - Code all day!
    Eat
    - Eat healthy food
    Sleep
    - Sleep soundly!
